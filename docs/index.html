Questions frequentes

Comment creer automatiquement un serveur Discord avec ce bot ?

Invite le bot sur un serveur Discord existant, puis lance /setup ou clique sur "Generer le serveur" depuis le dashboard web : les salons, roles, permissions et le reglement sont crees automatiquement en quelques secondes, a partir d'un template vivant.

Le bot Discord Serveur Creator est-il gratuit ?

Oui, l'utilisation de base (generation de serveur, moderation, roles de jeu, dashboard) est gratuite.

Comment fonctionnent les roles de jeu automatiques sur Discord ?

Le bot detecte les jeux joues par les membres via leur statut Discord et cree automatiquement un role et un salon dedies pour chaque jeu, sans configuration manuelle.

Quelles fonctionnalites de moderation sont incluses ?

Verification anti-bot (captcha), verification d'age a l'acceptation du reglement, detection anti-raid, logs d'audit et roles +16/-16 automatiques.

Peut-on gerer plusieurs serveurs Discord depuis un seul dashboard ?

Oui, le dashboard web liste tous les serveurs ou le bot est present et ou tu es administrateur, et permet de basculer entre eux sans se reconnecter.

Le bot propose-t-il un assistant IA ?

Oui, un assistant IA (Claude, GPT ou Gemini au choix) est integre au dashboard pour automatiser des taches de configuration par simple conversation.
